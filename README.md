# Souplesse Spec Kit

- Note: `node_modules` was removed from the repository and is ignored via `.gitignore`.

Setup
-----

Install dependencies and start the dev server:

```bash
npm install
npm run dev
```

Run tests:

```bash
npm test
```

If you accidentally committed large folders (like `node_modules`), they were removed from the repo history by untracking them; ensure you have a fresh `npm install` locally.
